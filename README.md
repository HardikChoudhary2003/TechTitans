# Healthcare Wellness Portal

A comprehensive healthcare platform with separate portals for patients and providers.

## Structure

- **frontend-patient**: React-based Patient UI
- **frontend-providerpublic**: React-based Provider & Public UI
- **backend-patientauth**: Node/Express Auth & Patient APIs
- **backend-providerpublic**: Node/Express Provider & Public APIs

## Tech Stack

- **Frontend**: React
- **Backend**: Node.js, Express
- **Database**: MongoDB (Mongoose) / SQL
- **Auth**: JWT, RBAC
